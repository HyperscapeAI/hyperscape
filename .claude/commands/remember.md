## Quick Reference

Store an explicit memory that you want Claude to remember for future sessions using mem0.ai integration.

### Usage
```bash
/remember [your note here]
```

This command will:
- Store your explicit memory immediately
- Make it available in future sessions
- Associate it with your user ID (git email)
- Enable context retrieval across conversations
